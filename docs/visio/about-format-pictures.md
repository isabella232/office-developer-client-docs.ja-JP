# <a name="about-format-pictures"></a>書式形式について

値の表示方法を指定するには、書式形式を使用します。たとえば、小数点の右側または左側に表示される数値の桁数や、文字列を大文字と小文字のどちらで表示するかを制御できます。
  
> [!NOTE]
> Microsoft Office system の書式設定を使用して日付と時刻の書式形式を定義するには、"{{m/d/yy}}" のように、書式形式を二重の中かっこで囲みます。 定義済みの形式 (201 など) を使用する場合は、中かっこと角かっこで囲みます。次のように"{ \<201\> }" 
  
次のセクションは、さまざまな種類の値を表示するための書式設定に使用できる記号を示しています。
  
## <a name="string-and-numeric-values"></a>文字列と数値

|**文字**|**説明**|
|:-----|:-----|
|#  <br/> |桁のプレースホルダーです。1 桁を表示するか、または何も表示しません。先行と後続のゼロは表示されません。小数点の左側にプレースホルダーの数を上回る桁がある場合、すべての桁が表示されます。小数点の右側にプレースホルダーの数を上回る桁がある場合、端数がプレースホルダーの数に合わせて四捨五入されます。寸法では、プレースホルダーが最も左側の桁である場合、0 のサブユニットは表示されません。  <br/> たとえば、FORMAT(0ft. 11.25in,"#.##u") は "11.25in." と表示されます。  <br/> |
|0  <br/> |桁のプレースホルダー (ゼロ) です。1 桁を表示するか、または何も表示しません。先行と後続のゼロは表示されます。小数点の左側にプレースホルダーの数を上回る桁がある場合、すべての桁が表示されます。小数点の右側にプレースホルダーの数を上回る桁がある場合、端数がプレースホルダーの数に合わせて四捨五入されます。寸法では、0 のサブユニットも表示されます。  <br/> たとえば、FORMAT(2ft. 11.33in,"0.## u") は "2 ft. 11.33 in." と表示されます。  <br/> |
|.  <br/> |小数点のプレースホルダーです。小数点の左側と右側に表示する桁数を指定します。複数の要素から構成される単位では、小数点は最小 (最も右側の) サブユニットで使用されます。コントロール パネルの [**地域と言語**] の設定で定義されている小数点の文字が表示されます。<br/> たとえば、FORMAT(250 cm,"0.000 u") は "250.000 cm" と表示されます。  <br/> |
|,  <br/> |桁区切り記号です。桁のプレースホルダー (# または 0) で囲まれた場合、小数点の左側に 4 桁以上の数値があると、桁区切り記号によって百の位と千の位が区切られます。コントロール パネルの [**地域と言語**] の設定で定義されている桁区切り記号が表示されます。<br/> |
|E- E+ e- e+  <br/> |科学的な書式です。書式で、これらの記号の右側に 1 つ以上の桁のプレースホルダーがある場合、数値は科学的な書式で表示されます。数値とその指数の間に、E または e が挿入されます。E+ または e+ を指定すると、正の指数の前にプラス (+) 記号が、負の指数の前にマイナス (-) 記号が表示されます。E- または e- を指定すると、指数が負の場合にのみマイナス (-) 記号が表示されます。  <br/> たとえば、FORMAT(12345.67,"###.#e+#") は ".123.5e+2" と表示されます。  <br/> |
|u または U  <br/> |短いラベルのプレースホルダーです。 各サブユニットの後に省略形の単位ラベルが挿入されます。 例: in.、ft.、deg.。 U のプレースホルダーでは大文字と小文字が混在するラベルが挿入され、u のプレースホルダーでは小文字のラベルが挿入されます。 プレースホルダーの前と同じ数のスペースがラベルの前に挿入されます。  <br/> たとえば、FORMAT(12 c 13 d,"#u") は、"13c1" と表示されます。  <br/> |
|uu または UU  <br/> |長いラベルのプレースホルダーです。 各サブユニットの後に単位ラベルが挿入されます。 例: インチ、フィート、度。U のプレースホルダーでは大文字と小文字が混在するラベルが挿入され、u のプレースホルダーでは小文字のラベルが挿入されます。 プレースホルダーの前に挿入するスペースと同じ数のスペースがラベルの前に挿入されます。  <br/> たとえば、FORMAT(12.43in,"# #/4 UU") は "12 2/4 INCHES" と表示されます。  <br/> |
|uuu または UUU  <br/> |汎用ラベルのプレース ホルダーです。 各サブユニットの後に汎用形式 (Visio では内部形式) の単位ラベルが挿入されます。 U のプレースホルダーでは大文字と小文字が混在するラベルが挿入され、u のプレースホルダーでは小文字のラベルが挿入されます。 プレースホルダーの前と同じ数のスペースがラベルの前に挿入されます。  <br/> |
|/  <br/> |分数のプレースホルダーです。このプレースホルダーを使用した書式の前に桁のプレースホルダーがある場合、式が整数と分数で表示されます。このような先行する桁のプレースホルダーがない場合は、分子に整数を配置した形式で表示されます。分母で桁のプレースホルダーの後に数値がある場合、分子が 1 になる最も近い分数に合わせて端数が四捨五入され、約分されます。桁のプレースホルダーを使用せずに分母に数値を指定した場合、最も近い分数に合わせて四捨五入されますが、約分はされません。  <br/> たとえば、FORMAT(12.43,"# #/4") は "12 2/4" と表示されます。  <br/> |
|スペース  <br/> |書式設定された出力に空白文字が表示されます。 別の文字を表示するには、円記号 (\)) 文字を使用します。  <br/> |
   
## <a name="currency-values"></a>通貨値

|**文字**|**説明**|
|:-----|:-----|
|$  <br/> |通貨記号です。コントロール パネルの [**地域と言語**] の設定で定義されている通貨記号が表示されます。<br/> |
|u または U  <br/> |短いラベルのプレースホルダーです。 現地通貨の標準の記号、または現地通貨以外の場合は 3 文字に省略された通貨単位が挿入されます。 たとえば、$99.00、42.70 FRF などです。 u のプレースホルダーでは小文字のラベルが挿入され、U では大文字と小文字が混在するラベルが挿入されます。  <br/> |
|uu または UU  <br/> |長いラベルのプレースホルダーです。 各サブユニットの後に長い通貨ラベルが挿入されます。 たとえば、U.S. dollar、French franc などです。 u のプレースホルダーでは小文字のラベルが挿入され、U では大文字と小文字が混在するラベルが挿入されます。  <br/> |
|uuu または UUU  <br/> |汎用ラベルのプレースホルダーです。 各サブユニットの後に、すべての通貨に対して、汎用の 3 文字の省略通貨単位が挿入されます。 たとえば、99.00 USD、42.70 FRF などです。 u のプレースホルダーでは小文字のラベルが挿入され、U では大文字と小文字が混在するラベルが挿入されます。 プレースホルダーの前と同じ数のスペースがラベルの前に挿入されます。  <br/> |
   
## <a name="text-values"></a>テキスト値

|**文字**|**説明**|
|:-----|:-----|
|\  <br/> |次の文字がそのまま表示されます。 円記号を表示するには、\\ と入力します。 「テキスト」も参照してください。  <br/> |
|"テキスト" または 'テキスト'  <br/> |引用符で囲まれたテキストがそのまま表示されます。「\ (円記号)」も参照してください。  <br/> |
|@  <br/> |テキストのプレースホルダーです。式の値が文字列の場合、文字列を置き換えます。  <br/> たとえば、FORMAT("Hello", "'You entered ('@')'" ) は "You entered (Hello)" と表示されます。  <br/> |
|@+  <br/> |大文字テキストのプレースホルダーです。入力した文字列の値を大文字に置き換えます。  <br/> たとえば、FORMAT("Hello", "@ @+ @-" ) は "Hello HELLO hello)" と表示されます。  <br/> |
|@-  <br/> |テキストのプレースホルダーです。 入力した文字列の値を小文字に置き換えます。  <br/> たとえば、FORMAT("Hello", "@ @+ @-" ) は "Hello HELLO hello)" と表示されます。  <br/> |
