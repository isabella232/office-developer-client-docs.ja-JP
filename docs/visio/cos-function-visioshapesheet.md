# <a name="cos-function-visioshapesheet"></a>COS 関数 (VisioShapeSheet)

角度のコサインを返します。 
  
## <a name="syntax"></a>構文

COS( *angle*) 
  
### <a name="parameters"></a>パラメーター

|**名前**|**必須 / オプション**|**データ型**|**説明**|
|:-----|:-----|:-----|:-----|
| _angle_ <br/> |必須かどうか  <br/> |**数値型 (Numeric)** <br/> |コサインを取得する角度を指定します。  <br/> |
   
## <a name="remarks"></a>注釈

COS 関数は、角度で別の測定単位を指定しない限り、ラジアンを使用  *します*  。 
  
## <a name="example"></a>例

COS(45 deg) 
  
0.7071 を返します。
