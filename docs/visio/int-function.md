# <a name="int-function"></a>INT 関数

数値を次の整数に切り上します。
  
## <a name="syntax"></a>構文

INT(** *number* ** ) 
  
### <a name="parameters"></a>パラメーター

|**名前**|**必須 / オプション**|**データ型**|**説明**|
|:-----|:-----|:-----|:-----|
| _number_ <br/> |必須かどうか  <br/> |**数値** <br/> |切り捨ての対象となる数値を指定します。  <br/> |
   
## <a name="example-1"></a>例 1

INT(7.2)
  
7 を返します。
  
## <a name="example-2"></a>例 2

INT(-7.2)
  
-8 を返します。
