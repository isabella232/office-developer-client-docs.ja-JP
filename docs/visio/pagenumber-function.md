# <a name="pagenumber-function"></a>PAGENUMBER 関数

図形を含むページの番号を返します。 
  
## <a name="syntax"></a>構文

PAGENUMBER()
  
### <a name="return-value"></a>戻り値

番号
  
## <a name="remarks"></a>注釈

現在のページが背景ページ (ページ番号なし) の場合、またはエラーが発生した場合は、PAGENUMBER 関数はゼロを返します。
