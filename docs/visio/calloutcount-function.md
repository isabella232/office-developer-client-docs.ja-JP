# <a name="calloutcount-function"></a>CALLOUTCOUNT 関数

図形に関連付けられた引き出し図形の合計数を返します。
  
## <a name="version-information"></a>バージョン情報

追加バージョン: Visio 2010
 
  
## <a name="syntax"></a>構文

CALLOUTCOUNT()
  
### <a name="return-value"></a>戻り値

 **Integer**
