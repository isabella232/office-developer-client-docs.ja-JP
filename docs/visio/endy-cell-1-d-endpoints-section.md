親図形  *の原点との*  関係で、1-D 図形の端点の y 座標を表します。

別の数式または **CellsU** プロパティを使用したプログラムから、名前によって [EndY] セルへの参照を取得するには、次の値を使用します。 
  
|||
|:-----|:-----|
| セル名 :  <br/> | EndY  <br/> |

|||
|:-----|:-----|
| セクション インデックス:  <br/> |**visSectionObject** <br/> |
