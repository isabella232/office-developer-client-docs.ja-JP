# <a name="data2-function"></a>DATA2 関数

図形の [図形名] ダイアログ ボックスの **[データ 2]** ボックスのテキストを文字列として返します ([開発] タブの [図形 **名]** をクリックします)。 
  
## <a name="syntax"></a>構文

DATA2 ()
  
## <a name="remarks"></a>注釈

ボックスが空の場合、関数は空の文字列を返します。
