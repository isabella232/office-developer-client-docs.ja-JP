# <a name="pnty-function"></a>PNTY 関数

ポイントの  _y_ 座標を返します。
  
## <a name="syntax"></a>構文

PNTX(** *point* ** ) 
  
### <a name="parameters"></a>パラメーター

|**名前**|**必須 / オプション**|**データ型**|**説明**|
|:-----|:-----|:-----|:-----|
| _point_ <br/> |必須かどうか  <br/> |**数値** <br/> |ポイント  _の y_ 座標。  <br/> |
   
### <a name="return-value"></a>戻り値

番号
  
## <a name="example"></a>例

PNTY(PNT(7,12)) 
  
12 を返します。
