# <a name="enter-an-empty-formula"></a>空の数式を入力する

1. 数式を含めるセルを選択します。
    
2. Del キーを押してから、Enter キーを押します。
    
> [!NOTE]
> ローカル数式を挿入した **後、Scratch** セルでこれを試してください。 マスターから継承する数式が存在しないので、セルは 'No Formula' に戻されます。
