# <a name="theme-function"></a>THEME 関数

現在のテーマの形式設定を取得します。
  
## <a name="syntax"></a>構文

THEME()
  
## <a name="example"></a>例

```vb
Shape.FillForegnd = THEME("FillColor")
```

現在のテーマの塗りつぶしの色を図形に割り当てます。
