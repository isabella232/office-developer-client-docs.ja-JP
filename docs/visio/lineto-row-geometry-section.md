[LineTo] 行には次のセルが含まれます。
  
|**Cell**|**説明**|
|:-----|:-----|
|[X](x-cell-geometry-section.md) <br/> |直線  *セグメント*  の終了頂点の x 座標。  <br/> |
|[Y](y-cell-geometry-section.md) <br/> |直線  *セグメントの*  終了頂点の y 座標。  <br/> |
