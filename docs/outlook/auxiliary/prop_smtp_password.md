# <a name="prop_smtp_password"></a>PROP_SMTP_PASSWORD

SMTP アカウントのパスワードを表します。
  
## <a name="quick-info"></a>クイック ヒント

|アイテム|コード|
|:-----|:-----|
|識別子:  <br/> |0x0205  <br/> |
|プロパティの種類:  <br/> |PT_UNICODE|SECURE_FLAG  <br/> |
|プロパティ タグ:  <br/> |0x0205801F  <br/> |
|アクセス:  <br/> |読み取り専用  <br/> |
