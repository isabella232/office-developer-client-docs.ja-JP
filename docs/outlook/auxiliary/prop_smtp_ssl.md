# <a name="prop_smtp_ssl"></a>PROP_SMTP_SSL

SMTP アカウントにセキュア ソケット層 (SSL) プロトコルを使用するかどうかを指定します。
  
## <a name="quick-info"></a>クイック ヒント

|||
|:-----|:-----|
|識別子:  <br/> |0x0202  <br/> |
|プロパティの種類:  <br/> |PT_DWORD  <br/> |
|プロパティ タグ:  <br/> |0x02020003  <br/> |
|アクセス:  <br/> |読み取り専用  <br/> |
   
## <a name="remarks"></a>注釈

値が 0 の場合は SSL 暗号化を使用しない、それ以外の場合は SSL 暗号化を使用します。
