# <a name="prop_smtp_use_auth"></a>PROP_SMTP_USE_AUTH

SMTP アカウントに認証を使用するかどうかを指定します。
  
## <a name="quick-info"></a>クイック ヒント

|||
|:-----|:-----|
|識別子:  <br/> |0x0203  <br/> |
|プロパティの種類:  <br/> |PT_DWORD  <br/> |
|プロパティ タグ:  <br/> |0x02030003  <br/> |
|アクセス:  <br/> |読み取り専用  <br/> |
   
## <a name="remarks"></a>注釈

値が 0 の場合は認証を使用しない、それ以外の場合は認証を使用します。
