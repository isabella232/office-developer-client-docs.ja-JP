# <a name="appointments"></a>予定

このセクションでは、予定に関連するサンプル タスクを示します。予定には、イベント、会議、定期的な予定などがあります。Outlook の予定表フォルダー内の予定は、[AppointmentItem](https://msdn.microsoft.com/library/bb645611\(v=office.15\)) オブジェクトで表されます。
