# <a name="pidnamelinecount-canonical-property"></a>PidNameLineCount 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
ドキュメントに添付されているファイル内の行数を指定します。
  
|||
|:-----|:-----|
|分名:  <br/> |なし  <br/> |
|プロパティ セット:  <br/> |PS_PUBLIC_STRINGS  <br/> |
|プロパティ名:  <br/> |LineCount  <br/> |
|データの種類 :   <br/> |PT_LONG  <br/> |
|エリア:  <br/> |共通  <br/> |
