# <a name="pidnamexsharingprovidername-canonical-property"></a>PidNameXSharingProviderName 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
**dispidSharingProviderGuid** ([PidLidSharingProviderGuid](pidlidsharingproviderguid-canonical-property.md)) プロパティで識別される共有プロバイダーのユーザー表示可能な名前を表します。
  
|||
|:-----|:-----|
|分名:  <br/> |なし  <br/> |
|プロパティ セット:  <br/> |PS_INTERNET_HEADERS  <br/> |
|プロパティ名:  <br/> |X-Sharing-Provider-Name  <br/> |
|データの種類 :   <br/> |PT_UNICODE  <br/> |
|エリア:  <br/> |共有  <br/> |
