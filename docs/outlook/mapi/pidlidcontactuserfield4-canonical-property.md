# <a name="pidlidcontactuserfield4-canonical-property"></a>PidLidContactUserField4 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
連絡先のその他の用途を対象とした連絡先の 4 番目のフィールドを指定します。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |dispidContactUserField4  <br/> |
|プロパティ セット:  <br/> |PSETID_Address  <br/> |
|長い ID (LID):  <br/> |0x00008052  <br/> |
|データの種類 :   <br/> |PT_UNICODE  <br/> |
|エリア:  <br/> |Contact  <br/> |
   
## <a name="remarks"></a>注釈

このプロパティには、他の連絡先固有のプロパティとは無関係のテキストが含まれる。
