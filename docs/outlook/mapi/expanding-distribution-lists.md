# <a name="expanding-distribution-lists"></a>配布リストの展開

  
  
**適用対象**: Outlook 
  
 **配布リストを展開するための MAPI メッセージを表示するのには**
  
- MAPIPDL に、 **PR_ADDRTYPE** ([PidTagAddressType](pidtagaddresstype-canonical-property.md)) のプロパティを設定します。
    
    MAPI は、トランスポート プロバイダーにメッセージを送信する前にこのタイプのアドレスを展開します。
