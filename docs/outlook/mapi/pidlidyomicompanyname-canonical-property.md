# <a name="pidlidyomicompanyname-canonical-property"></a>PidLidYomiCompanyName 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
連絡先の会社名の発音を表す音を指定します。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |dispidYomiCompanyName  <br/> |
|プロパティ セット:  <br/> |PSETID_Address  <br/> |
|長い ID (LID):  <br/> |0x0000802E  <br/> |
|データの種類 :   <br/> |PT_UNICODE  <br/> |
|エリア:  <br/> |Contact  <br/> |
