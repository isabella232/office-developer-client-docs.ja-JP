# <a name="pidlidfax3addresstype-canonical-property"></a>PidLidFax3AddressType 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
他の連絡先の FAX アドレスのアドレスの種類を指定します。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |dispidFax3AddrType  <br/> |
|プロパティ セット:  <br/> |PSETID_Address  <br/> |
|長い ID (LID):  <br/> |0x000080D2  <br/> |
|データの種類 :   <br/> |PT_UNICODE  <br/> |
|エリア:  <br/> |Contact  <br/> |
   
## <a name="remarks"></a>注釈

このプロパティが存在する場合は、"FAX" に設定する必要があります。
