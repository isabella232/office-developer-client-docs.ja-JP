# <a name="pidtagmessagesizeextended-canonical-property"></a>PidTagMessageSizeExtended 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
64 ビット整数バージョンの PR_MESSAGE_SIZE **(** [PidTagMessageSize](pidtagmessagesize-canonical-property.md)) プロパティを含む。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_MESSAGE_SIZE_EXTENDED  <br/> |
|識別子:  <br/> |0x0E08  <br/> |
|データの種類 :   <br/> |PT_I8  <br/> |
|エリア:  <br/> |一般的なメッセージング  <br/> |
