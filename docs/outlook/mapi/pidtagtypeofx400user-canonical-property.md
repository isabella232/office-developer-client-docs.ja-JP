# <a name="pidtagtypeofx400user-canonical-property"></a>PidTagTypeOfX400User 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
このプロパティには、レポートで使用するメッセージ受信者の種類が含まれる。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_TYPE_OF_MTS_USER  <br/> |
|識別子:  <br/> |0x0C1C  <br/> |
|データの種類 :   <br/> |PT_LONG  <br/> |
|エリア:  <br/> |MAPI 受信者  <br/> |
