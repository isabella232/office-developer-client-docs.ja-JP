# <a name="pidlidreferenceentryid-canonical-property"></a>PidLidReferenceEntryId 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
連絡先の参照 [ENTRYID](entryid.md) を指定します。 
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |dispidReferenceEID  <br/> |
|プロパティ セット:  <br/> |PSETID_Common  <br/> |
|長い ID (LID):  <br/> |0x000085BD  <br/> |
|データの種類 :   <br/> |PT_BINARY  <br/> |
|エリア:  <br/> |Contact  <br/> |
   
## <a name="remarks"></a>注釈

存在する場合、このプロパティは連絡先の **EntryId の値と** 等しい必要があります。
