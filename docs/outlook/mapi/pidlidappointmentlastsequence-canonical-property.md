# <a name="pidlidappointmentlastsequence-canonical-property"></a>PidLidAppointmentLastSequence 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
出席者に送信された最後のシーケンス番号を開催者に示します。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |dispidApptLastSequence  <br/> |
|プロパティ セット:  <br/> |PSETID_Appointment  <br/> |
|長い ID (LID):  <br/> |0x00008203  <br/> |
|データの種類 :   <br/> |PT_LONG  <br/> |
|エリア:  <br/> |会議  <br/> |
   
## <a name="remarks"></a>注釈

このプロパティは、出席者には意味はありません。
