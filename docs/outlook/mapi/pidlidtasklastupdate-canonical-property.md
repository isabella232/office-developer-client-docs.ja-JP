# <a name="pidlidtasklastupdate-canonical-property"></a>PidLidTaskLastUpdate 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
タスクに対して行われた **、dispidTaskHistory** ([PidLidTaskHistory](pidlidtasklastupdate-canonical-property.md)) プロパティによって示された最新の変更の協定世界時 (UTC) と日付を示します。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |dispidTaskLastUpdate  <br/> |
|プロパティ セット:  <br/> |PSETID_Task  <br/> |
|長い ID (LID):  <br/> |0x00008115  <br/> |
|データの種類 :   <br/> |PT_SYSTIME  <br/> |
|エリア:  <br/> |タスク  <br/> |
