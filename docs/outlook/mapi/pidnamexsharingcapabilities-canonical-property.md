# <a name="pidnamexsharingcapabilities-canonical-property"></a>PidNameXSharingCapabilities 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
アイテムの共有機能を示す値を含む。
  
|||
|:-----|:-----|
|分名:  <br/> |なし  <br/> |
|プロパティ セット:  <br/> |PS_INTERNET_HEADERS  <br/> |
|プロパティ名:  <br/> |X-Sharing-Capabilities  <br/> |
|データの種類 :   <br/> |PT_UNICODE  <br/> |
|エリア:  <br/> |共有  <br/> |
   
## <a name="remarks"></a>注釈

このプロパティには、メッセージ オブジェクトの共有機能を示す値が含まれる。
