# <a name="cbmtsid"></a>CbMTSID

  
  
**適用対象**: Outlook 2013 | Outlook 2016

|||
|:-----|:-----|
|ヘッダー ファイル:  <br/> |Mapidefs.h  <br/> |
|関連する構造:  <br/> |**MTSID** <br/> |
   
```cpp
CbMTSID (_lpentry)
```

## <a name="parameters"></a>パラメーター

 _ _lpentry_
  
> 既存の **MTSID 構造への** ポインター。
