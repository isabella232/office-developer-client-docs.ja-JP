# <a name="pidtagservertypedisplayname-canonical-property"></a>PidTagServerTypeDisplayName 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016

|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_SERVER_TYPE_DISPLAY_NAME  <br/> |
|識別子:  <br/> |0x341D  <br/> |
|データの種類 :   <br/> |PT_BINARY  <br/> |
|エリア:  <br/> |共通  <br/> |
