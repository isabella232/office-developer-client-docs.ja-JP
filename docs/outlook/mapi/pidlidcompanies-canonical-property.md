# <a name="pidlidcompanies-canonical-property"></a>PidLidCompanies 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
アイテムに関連付けられている会社の名前を格納します。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |dispidCompanies  <br/> |
|プロパティ セット:  <br/> |PSETID_Common  <br/> |
|長い ID (LID):  <br/> |0x00008539  <br/> |
|データの種類 :   <br/> |PT_MV_UNICODE  <br/> |
|エリア:  <br/> |一般的なメッセージング  <br/> |
