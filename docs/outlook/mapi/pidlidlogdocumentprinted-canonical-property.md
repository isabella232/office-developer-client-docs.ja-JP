# <a name="pidlidlogdocumentprinted-canonical-property"></a>PidLidLogDocumentPrinted 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
ジャーナリング中にドキュメントが印刷されたかどうかを示します。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |dispidLogDocPrinted  <br/> |
|プロパティ セット:  <br/> |PSETID_Log  <br/> |
|長い ID (LID):  <br/> |0x0000870E  <br/> |
|データの種類 :   <br/> |PT_BOOLEAN  <br/> |
|エリア:  <br/> |ジャーナル  <br/> |
