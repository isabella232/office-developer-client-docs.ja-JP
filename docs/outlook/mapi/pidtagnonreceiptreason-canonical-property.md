# <a name="pidtagnonreceiptreason-canonical-property"></a>PidTagNonReceiptReason 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
配信不可レポートの一部を形成するメッセージが受信されない理由が含まれる。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_NON_RECEIPT_REASON  <br/> |
|識別子:  <br/> |0x003E  <br/> |
|データの種類 :   <br/> |PT_LONG  <br/> |
|エリア:  <br/> |Exchange  <br/> |
   
## <a name="remarks"></a>注釈

理由としては、メッセージが破棄されたなどの理由があります。
