# <a name="pidtagprofiletype-canonical-property"></a>PidTagProfileType 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
情報ストアの種類を格納します。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_PROFILE_TYPE  <br/> |
|識別子:  <br/> |0x660A  <br/> |
|データの種類 :   <br/> |PT_LONG  <br/> |
|エリア:  <br/> |MAPI プロファイルの構成  <br/> |
   
## <a name="remarks"></a>注釈

このプロパティに指定可能な値は次のとおりです。
  
- 1 - プライマリ ユーザー
    
- 2 - デリゲート
    
- 3 - パブリック ストア
    
- 4 - サブスクリプション
