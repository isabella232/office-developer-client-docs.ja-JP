# <a name="pidtagspamtrustedsenders-canonical-property"></a>PidTagSpamTrustedSenders 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
信頼できる送信者で構成される電子メール アドレスとドメインのセミコロンで区切られたリストが含まれます。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_SPAM_TRUSTED_SENDERS_W  <br/> |
|長い ID (LID):  <br/> |0x0418  <br/> |
|データの種類 :   <br/> |PT_UNICODE  <br/> |
|エリア:  <br/> |スパム  <br/> |
