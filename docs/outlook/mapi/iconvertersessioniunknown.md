# <a name="iconvertersession--iunknown"></a>IConverterSession : IUnknown

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
MIME オブジェクトと MAPI メッセージ間の変換を許可します。 これは、インターネットを通してメッセージを転送する場合に役立ちます。
