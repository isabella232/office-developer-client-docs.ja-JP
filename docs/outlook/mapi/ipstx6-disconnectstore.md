# <a name="ipstx6disconnectstore"></a>IPSTX6::DisconnectStore

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
ローカル ストアを切断します。
  
```cpp
HRESULT DisconnectStore( 
    ULONG ulFlags 
);
```

## <a name="parameters"></a>パラメーター

 _ulFlags_
  
>  [in]動作を変更するフラグです。 このパラメーターは常に 0 に設定する必要があります。
