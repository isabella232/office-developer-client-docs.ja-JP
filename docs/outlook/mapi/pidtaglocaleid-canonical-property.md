# <a name="pidtaglocaleid-canonical-property"></a>PidTagLocaleId 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
配信レポートなど、システムによって生成されたメッセージを翻訳するための言語ロケールを確立します。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_LOCALE_ID  <br/> |
|識別子:  <br/> |0x66A1  <br/> |
|データの種類 :   <br/> |PT_LONG  <br/> |
|エリア:  <br/> |その他  <br/> |
   
## <a name="remarks"></a>注釈

このプロパティは、プライベート メールボックス ログオンで使用できます。
