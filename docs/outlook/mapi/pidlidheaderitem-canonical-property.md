# <a name="pidlidheaderitem-canonical-property"></a>PidLidHeaderItem 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
Microsoft Outlook がメッセージのヘッダーのみ、またはメッセージのヘッダーと本文の両方をクライアント コンピューターにダウンロードしたかどうかを指定します(Outlook がキャッシュ Exchange モードで構成されている場合)。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |dispidHeaderItem  <br/> |
|プロパティ セット:  <br/> |PSETID_Common  <br/> |
|長い ID (LID):  <br/> |0x00008578  <br/> |
|データの種類 :   <br/> |PT_LONG  <br/> |
|エリア:  <br/> |一般的なメッセージング  <br/> |
