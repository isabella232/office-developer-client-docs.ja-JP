# <a name="pidlidworkaddressstreet-canonical-property"></a>PidLidWorkAddressStreet 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
連絡先の仕事用郵送先住所の通りの部分を指定します。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |dispidWorkAddressStreet  <br/> |
|プロパティ セット:  <br/> |PSETID_Address  <br/> |
|長い ID (LID):  <br/> |0x00008045  <br/> |
|データの種類 :   <br/> |PT_UNICODE  <br/> |
|エリア:  <br/> |Contact  <br/> |
