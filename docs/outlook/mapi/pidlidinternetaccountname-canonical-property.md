# <a name="pidlidinternetaccountname-canonical-property"></a>PidLidInternetAccountName 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
電子メール メッセージを送信するユーザーに表示される電子メール アカウント名を指定します。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |dispidInetAcctName  <br/> |
|プロパティ セット:  <br/> |PSETID_Common  <br/> |
|長い ID (LID):  <br/> |0x00008580  <br/> |
|データの種類 :   <br/> |PT_UNICODE  <br/> |
|エリア:  <br/> |一般的なメッセージング  <br/> |
   
## <a name="remarks"></a>注釈

この文字列の形式は実装に依存します。 このプロパティは、クライアントがメールを送信するサーバーを決定するために使用できますが、オプションであり、値はサーバーに対して意味を持つ必要はありません。
