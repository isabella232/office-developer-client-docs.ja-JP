# <a name="pidtagweddinganniversary-canonical-property"></a>PidTagWeddingAnniversary 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
ユーザーの結婚記念日の日付が含まれる。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_WEDDING_ANNIVERSARY  <br/> |
|識別子:  <br/> |0x3A41  <br/> |
|データの種類 :   <br/> |PT_SYSTIME  <br/> |
|エリア:  <br/> |MAPI メール ユーザー  <br/> |
   
## <a name="remarks"></a>注釈

このプロパティは、ユーザーまたはユーザーの組織によって定義されます。
