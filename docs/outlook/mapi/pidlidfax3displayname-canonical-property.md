# <a name="pidlidfax3displayname-canonical-property"></a>PidLidFax3DisplayName 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
連絡先の他の FAX アドレスの表示名を指定します。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |dispidFax3DisplayName  <br/> |
|プロパティ セット:  <br/> |PSETID_Address  <br/> |
|長い ID (LID):  <br/> |0x000080D0  <br/> |
|データの種類 :   <br/> |PT_UNICODE  <br/> |
|エリア:  <br/> |Contact  <br/> |
