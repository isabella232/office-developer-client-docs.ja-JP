# <a name="pidtagbusiness2telephonenumbers-canonical-property"></a>PidTagBusiness2TelephoneNumbers 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
ビジネスの電話番号の一覧を含む。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_BUSINESS2_TELEPHONE_NUMBER_A_MV  <br/> |
|識別子:  <br/> |0x3A1B  <br/> |
|データの種類 :   <br/> |PT_MV_STRING8  <br/> |
|エリア:  <br/> |Contact  <br/> |
   
## <a name="remarks"></a>注釈

このプロパティを使用して、代替電話番号またはビジネスの内線番号の一覧を一覧表示できます。
