# <a name="pidtaghome2telephonenumbers-canonical-property"></a>PidTagHome2TelephoneNumbers 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016

|||

|関連するプロパティ:  <br/> |PR_HOME2_TELEPHONE_NUMBER_A_MV  <br/> |

|データの種類 :   <br/> |PT_MV_UNICODE8  <br/> |
