# <a name="pidlidsharingprovidername-canonical-property"></a>PidLidSharingProviderName 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
**dispidSharingProviderGuid** ([PidLidSharingProviderGuid)](pidlidsharingproviderguid-canonical-property.md)で識別される共有プロバイダーのユーザー表示可能な名前を指定します。 これは、共有メッセージのプロパティです。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |dispidSharingProviderName  <br/> |
|プロパティ セット:  <br/> |PSETID_Sharing  <br/> |
|長い ID (LID):  <br/> |0x00008A02  <br/> |
|データの種類 :   <br/> |PT_UNICODE  <br/> |
|エリア:  <br/> |共有  <br/> |
   
## <a name="remarks"></a>注釈

このプロパティは無視する必要があります。
