# <a name="pidlidtaskrecurrence-canonical-property"></a>PidLidTaskRecurrence 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
定期的なタスクに関する情報を提供する RecurrencePattern 構造が含まれています。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |dispidTaskRecur  <br/> |
|プロパティ セット:  <br/> |PSETID_Task  <br/> |
|長い ID (LID):  <br/> |0x00008116  <br/> |
|データの種類 :   <br/> |PT_BINARY  <br/> |
|エリア:  <br/> |タスク  <br/> |
   
## <a name="remarks"></a>注釈

RecurrencePattern 構造を作成して指定する方法の詳細については、「Create a Simple Recurrent Task Item 」 [を参照してください](how-to-create-a-simple-recurrent-task-item.md)。
