# <a name="pidlidoldwhenstartwhole-canonical-property"></a>PidLidOldWhenStartWhole 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
会議の更新前に **dispidApptStartWhole** [(PidLidAppointmentStartWhole)](pidlidappointmentstartwhole-canonical-property.md)プロパティの元の値を示します。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |dispidOldWhenStartWhole  <br/> |
|プロパティ セット:  <br/> |PSETID_Meeting  <br/> |
|長い ID (LID):  <br/> |0x00000029  <br/> |
|データの種類 :   <br/> |PT_SYSTIME  <br/> |
|エリア:  <br/> |会議  <br/> |
   
## <a name="remarks"></a>注釈

このプロパティは必須ではありません。
