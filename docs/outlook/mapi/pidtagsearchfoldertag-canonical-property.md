# <a name="pidtagsearchfoldertag-canonical-property"></a>PidTagSearchFolderTag 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
この定義メッセージを一致する検索フォルダー コンテナーと同期するために使用される値を格納します。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_WB_SF_TAG  <br/> |
|識別子:  <br/> |0x6847  <br/> |
|データの種類 :   <br/> |PT_LONG  <br/> |
|エリア:  <br/> |検索  <br/> |
   
## <a name="remarks"></a>注釈

このプロパティは、定義メッセージが変更された場合に変更されます。 繰り返しごとに変更する必要がありますが、一意ではない場合があります。
