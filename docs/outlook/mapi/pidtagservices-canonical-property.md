# <a name="pidtagservices-canonical-property"></a>PidTagServices 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
現在のプロファイル内のメッセージ サービスの識別子の一覧を含む。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_SERVICES  <br/> |
|識別子:  <br/> |0x3D0E  <br/> |
|データの種類 :   <br/> |PT_BINARY  <br/> |
|エリア:  <br/> |MAPI プロファイル  <br/> |
   
## <a name="remarks"></a>注釈

このプロパティを使用しません。 MAPI で使用するために予約されています。
