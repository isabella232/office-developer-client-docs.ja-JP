# <a name="pidtagsearchfolderid-canonical-property"></a>PidTagSearchFolderId 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
検索フォルダーを識別するグローバル一意識別子 (GUID) が含まれる。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_WB_SF_ID  <br/> |
|識別子:  <br/> |0x6842  <br/> |
|データの種類 :   <br/> |PT_BINARY  <br/> |
|エリア:  <br/> |検索  <br/> |
