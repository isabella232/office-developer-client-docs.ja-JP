# <a name="pidlidcommonstart-canonical-property"></a>PidLidCommonStart 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
メッセージの開始日時を表します。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |dispidCommonStart  <br/> |
|プロパティ セット:  <br/> |PSETID_Common  <br/> |
|長い ID (LID):  <br/> |0x00008516  <br/> |
|データの種類 :   <br/> |PT_SYSTIME  <br/> |
|エリア:  <br/> |一般的なメッセージング  <br/> |
   
## <a name="remarks"></a>注釈

このプロパティは、アイテムの開始時刻を示します。 **dispidCommonEnd** ([PidLidCommonEnd](pidlidcommonend-canonical-property.md)) プロパティの値以下である必要があります。
  
このプロパティの値は **、dispidTaskStartDate** ([PidLidTaskStartDate](pidlidtaskstartdate-canonical-property.md)) プロパティと同等の協定世界時 (UTC) である必要があります。
