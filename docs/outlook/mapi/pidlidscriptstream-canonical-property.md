# <a name="pidlidscriptstream-canonical-property"></a>PidLidScriptStream 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
メッセージに関連付けられたカスタム フォームVisual Basicスクリプト エディション (VBScript) で記述されたコードを表します。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |dispidScriptStream  <br/> |
|プロパティ セット:  <br/> |PSETID_Common  <br/> |
|長い ID (LID):  <br/> |0x00008541  <br/> |
|データの種類 :   <br/> |PT_BINARY  <br/> |
|エリア:  <br/> |実行時の構成  <br/> |
