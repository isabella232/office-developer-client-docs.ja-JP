# <a name="pidlidtaskcomplete-canonical-property"></a>PidLidTaskComplete 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016

|||
|:-----|:-----|
|関連するプロパティ:  <br/> |dispidTaskComplete  <br/> |
|プロパティ セット:  <br/> |PSETID_Task  <br/> |

|データの種類 :   <br/> |PT_BOOLEAN  <br/> |
