# <a name="pidtagrpcoverhttpproxyserver-canonical-property"></a>PidTagRpcOverHttpProxyServer 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
HTTP プロトコルまたは URL の前にある Hypertext Transfer Protocol Secure (HTTPS) プロトコルを使用せずに Microsoft Outlook から Microsoft Exchange Server に接続するために使用されるプロキシ サーバーの URL を格納します。
  
## 

|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_ROH_PROXY_SERVER  <br/> |
|識別子:  <br/> |0x6622  <br/> |
|プロパティの種類:  <br/> |PT_UNICODE  <br/> |
|エリア:  <br/> |その他  <br/> |
