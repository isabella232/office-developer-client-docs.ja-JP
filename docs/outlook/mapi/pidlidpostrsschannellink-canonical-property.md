# <a name="pidlidpostrsschannellink-canonical-property"></a>PidLidPostRssChannelLink 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
XML ファイルが送信された RSS または Atom フィードの URL が含まれる。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |dispidPostRssChannelLink  <br/> |
|プロパティ セット:  <br/> |PSETID_PostRss  <br/> |
|長い ID (LID):  <br/> |0x00008900  <br/> |
|データの種類 :   <br/> |PT_UNICODE  <br/> |
|エリア:  <br/> |RSS  <br/> |
