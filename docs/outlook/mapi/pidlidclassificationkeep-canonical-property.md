# <a name="pidlidclassificationkeep-canonical-property"></a>PidLidClassificationKeep 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
メッセージが転送または返信された場合に、メッセージの分類をメッセージと一緒に保持するかどうかを示します。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |dispidClassKeep  <br/> |
|プロパティ セット:  <br/> |PSETID_Common  <br/> |
|長い ID (LID):  <br/> |0x000085BA  <br/> |
|データの種類 :   <br/> |PT_BOOLEAN  <br/> |
|エリア:  <br/> |一般的なメッセージング  <br/> |
   
## <a name="remarks"></a>注釈

詳細については、「How to Deploy Message [Classification for Outlook 2007」を参照](https://msdn.microsoft.com/library/5a220424-edd5-4a21-b7fd-8106c23c3b39.aspx)してください。
