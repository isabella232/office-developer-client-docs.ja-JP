# <a name="pidtagotheraddressstateorprovince-canonical-property"></a>PidTagOtherAddressStateOrProvince 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
他のアドレスで使用される州または州の名前を含む。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_OTHER_ADDRESS_STATE_OR_PROVINCE、PR_OTHER_ADDRESS_STATE_OR_PROVINCE_A、PR_OTHER_ADDRESS_STATE_OR_PROVINCE_W  <br/> |
|識別子:  <br/> |0x3A62  <br/> |
|データの種類 :   <br/> |PT_UNICODE、PT_STRING8  <br/> |
|エリア:  <br/> |Address  <br/> |
   
## <a name="remarks"></a>注釈

これらのプロパティは、ユーザーまたはユーザーの組織によって定義されます。
