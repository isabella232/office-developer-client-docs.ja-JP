# <a name="pidtagpreprocess-canonical-property"></a>PidTagPreprocess 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
メッセージに前処理が必要な場合は TRUE を含む。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_PREPROCESS  <br/> |
|識別子:  <br/> |0x0E22  <br/> |
|データの種類 :   <br/> |PT_BOOLEAN  <br/> |
|エリア:  <br/> |MAPI 送信不可  <br/> |
