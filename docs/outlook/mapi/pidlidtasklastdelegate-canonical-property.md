# <a name="pidlidtasklastdelegate-canonical-property"></a>PidLidTaskLastDelegate 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
 最近割り当てられたユーザー、またはタスクが割り当てられたユーザーの名前を指定します。 
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |dispidTaskLastDelegate  <br/> |
|プロパティ セット:  <br/> |PSETID_Task  <br/> |
|長い ID (LID):  <br/> |0x00008125  <br/> |
|データの種類 :   <br/> |PT_UNICODE  <br/> |
|エリア:  <br/> |タスク  <br/> |
   
## <a name="remarks"></a>注釈

タスク要求を送信する前に、クライアントはタスク割り当て者の名前にこのプロパティを設定します。 タスク応答を送信する前に、クライアントはタスクの割り当て先の名前にこのプロパティを設定します。
