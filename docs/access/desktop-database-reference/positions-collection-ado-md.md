# <a name="positions-collection-ado-md"></a>Positions コレクション (ADO MD)


**適用先:** Access 2013、Office 2013

軸上にあるポイントを定義する [Position](position-object-ado-md.md) オブジェクトを含みます。

## <a name="remarks"></a>注釈

**Positions** は、標準の ADO コレクションです。このコレクションのプロパティとメソッドを使用すると、次の操作を実行できます。

  - [Count](count-property-ado.md) プロパティを使用して、コレクションのオブジェクトの数を取得します。

  - 既定の [Item](item-property-ado.md) プロパティを使用して、コレクションからオブジェクトを返します。

  - [Refresh](refresh-method-ado.md) メソッドを使用して、プロバイダーからコレクションのオブジェクトを更新します。
