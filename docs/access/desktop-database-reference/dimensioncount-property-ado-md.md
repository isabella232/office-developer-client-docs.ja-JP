# <a name="dimensioncount-property-ado-md"></a>DimensionCount プロパティ (ADO MD)


**適用先**: Access 2013、Office 2013

軸の次元の数を示します。

## <a name="return-values"></a>戻り値

長整数型 ( **Long** ) の値を取得します。値の取得のみが可能です。
