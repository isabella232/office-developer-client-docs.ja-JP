# <a name="databasetransactions-property-dao"></a>Database.Transactions プロパティ (DAO)


**適用先**: Access 2013、Office 2013

オブジェクトがトランザクションをサポートしているかどうかを示す値を取得します。値の取得のみ可能です。ブール型 ( **Boolean**) の値を使用します。

## <a name="syntax"></a>構文

*式* .トランザクション

*式* **Database** オブジェクトを表す変数です。
