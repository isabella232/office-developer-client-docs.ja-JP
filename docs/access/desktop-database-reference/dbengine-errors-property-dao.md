# <a name="dbengineerrors-property-dao"></a>DBEngine.Errors プロパティ (DAO)


**適用先:** Access 2013、Office 2013

指定したオブジェクトの、すべての保存された **Error** オブジェクトを含む **Errors** コレクションを返します。値の取得のみ可能です。

## <a name="syntax"></a>構文

*式* .エラー

*式* **DBEngine** オブジェクトを表す変数です。
