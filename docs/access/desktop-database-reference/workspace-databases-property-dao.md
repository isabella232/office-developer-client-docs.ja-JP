# <a name="workspacedatabases-property-dao"></a>Databases プロパティ (DAO)


**適用先:** Access 2013、Office 2013

指定された **Workspace** の開いているデータベースを表す **Databases** コレクションを取得します。 読み取り専用です。

## <a name="syntax"></a>構文

*式*。Databases

*式***Workspace**オブジェクトを表す変数を取得します。
