# <a name="databaseupdatable-property-dao"></a>Database.Updatable プロパティ (DAO)


**適用先**: Access 2013、Office 2013

DAO オブジェクトを変更できるかどうかを示す値を取得します。値の取得のみ可能です。ブール型 ( **Boolean** ) の値を使用します。  

## <a name="syntax"></a>構文

*式* .更新可能

*式* **Database** オブジェクトを表す変数です。
