# <a name="indexes-members-dao"></a>インデックス メンバー (DAO)


**適用先**: Access 2013、Office 2013

Indexes コレクションには、格納されているすべての TableDef オブジェクトの Index オブジェクトが含まれます (Microsoft Access ワークスペースのみ)。
