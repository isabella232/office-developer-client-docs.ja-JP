# <a name="fieldstatusenum"></a>FieldStatusEnum

**適用先:** Access 2013、Office 2013

**Field** オブジェクトの状態を表します。

**adFieldPending\*** 値は、状態を設定する原因となった操作を示し、他の状態値と組み合わせて使用される場合があります。
