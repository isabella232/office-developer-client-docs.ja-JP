# <a name="propertiesdelete-method-dao"></a>Properties.Delete メソッド (DAO)

**適用先:** Access 2013、Office 2013

指定した **Property** オブジェクトを **Properties** コレクションから削除します。

## <a name="syntax"></a>構文

*式* .Delete(***Name***)

*式* Properties オブジェクトを表す **変数** 。
