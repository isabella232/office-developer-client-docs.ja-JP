# <a name="databasecontainers-property-dao"></a>Database.Containers プロパティ (DAO)


**適用先:** Access 2013、Office 2013

指定したデータベースのすべての **Container** オブジェクトを表す **Containers** コレクションを返します。値の取得のみ可能です。

## <a name="syntax"></a>構文

*式* .コンテナー

*式* **Database** オブジェクトを表す変数です。
