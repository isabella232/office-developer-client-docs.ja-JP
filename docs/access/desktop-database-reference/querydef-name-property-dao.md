# <a name="querydefname-property-dao"></a>QueryDef.Name プロパティ (DAO)


**適用先:** Access 2013、Office 2013

指定したオブジェクトの名前を設定または返します。 読み取りまたは書き込み **文字列** です。

## <a name="syntax"></a>構文

*式*。拡張子

*式***QueryDef**オブジェクトを表す変数を取得します。

## <a name="remarks"></a>注釈

**QueryDef** オブジェクトの名前の最大長は 64 文字です。
