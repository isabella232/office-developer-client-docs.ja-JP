# <a name="databasetabledefs-property-dao"></a>Database.TableDefs プロパティ (DAO)


**適用先**: Access 2013、Office 2013

指定したデータベースに保存されたすべての **TableDef** オブジェクトを含む **TableDefs** コレクションを返します。値の取得のみ可能です。

## <a name="syntax"></a>構文

*式* .TableDefs

*式* **Database** オブジェクトを表す変数です。
