# <a name="tabledefname-property-dao"></a>TableDef.Name プロパティ (DAO)


**適用先**: Access 2013、Office 2013

指定したオブジェクトの名前を設定または返します。読み取りまたは書き込み **文字列** です。

## <a name="syntax"></a>構文

*式* .名前

*式***TableDef** オブジェクトを表す変数。

## <a name="remarks"></a>注釈

**TableDef** オブジェクトの名前の最大長は 64 文字です。
