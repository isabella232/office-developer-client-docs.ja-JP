# <a name="databasequerydefs-property-dao"></a>Database.QueryDefs プロパティ (DAO)


**適用先:** Access 2013、Office 2013

指定したデータベースのすべての **QueryDef** オブジェクトを含む **QueryDefs** コレクションを返します。値の取得のみ可能です。

## <a name="syntax"></a>構文

*式* .QueryDefs

*式* **Database** オブジェクトを表す変数です。
