# <a name="connectionname-property-dao"></a>Connection.Name プロパティ (DAO)


**適用先**: Access 2013、Office 2013

**[Connection](connection-object-dao.md)** オブジェクトの名前を取得します。

## <a name="syntax"></a>構文

*式* .名前

*式***Connection** オブジェクトを表す変数です。

## <a name="remarks"></a>注釈

コード内の Visual Basic for Applications の **Dim** ステートメントでオブジェクトの **Name** プロパティを使用すると、そのオブジェクトの別のインスタンスを作成できます。
