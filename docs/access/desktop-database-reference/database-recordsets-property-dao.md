# <a name="databaserecordsets-property-dao"></a>Database.Recordsets プロパティ (DAO)


**適用先**: Access 2013、Office 2013

指定したデータベースの開いているレコードセットをすべて含む **Recordsets** コレクションを返します。値の取得のみ可能です。

## <a name="syntax"></a>構文

*式* .Recordsets

*式* **Database** オブジェクトを表す変数です。
