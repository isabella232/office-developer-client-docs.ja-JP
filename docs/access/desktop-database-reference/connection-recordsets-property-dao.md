# <a name="connectionrecordsets-property-dao"></a>Connection.Recordsets プロパティ (DAO)


**適用先**: Access 2013、Office 2013

指定した接続の開いているレコードセットをすべて含む **Recordsets** コレクションを返します。値の取得のみ可能です。

## <a name="syntax"></a>構文

*式* .Recordsets

*式***Connection** オブジェクトを表す変数です。
