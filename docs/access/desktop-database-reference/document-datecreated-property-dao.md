# <a name="documentdatecreated-property-dao"></a>Document.DateCreated プロパティ (DAO)


**適用先:** Access 2013、Office 2013

オブジェクトが作成された日時を返します。値の取得のみ可能です。バリアント型 ( **Variant**) の値を使用します。

## <a name="syntax"></a>構文

*式* .DateCreated

*式* Document オブジェクトを表す **変数** 。
